# 🖥️​ Vlog-space 

Una plataforma web moderna de tipo **Vlog** diseñada para la visulizacion de mi portafolio como desarrollador web. 
Este proyecto ha sido desarrollado con un enfoque híbrido, combinando un diseño de interfaz intuitivo y responsivo en el Frontend con una arquitectura lógica sólida en el Backend.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** HTML5, CSS3, JavaScript.
*   **Backend & Lógica:** Python.
*   **Control de Versiones:** Git y GitHub.

## 🚀 Características Principales

*   **Diseño Responsivo:** Interfaz adaptada para una navegación óptima tanto en dispositivos móviles como en computadoras de escritorio.
*   **Gestión de Posts:** Sistema estructurado para la lectura, publicación y organización de entradas de vlog y contenido multimedia.
*   **Arquitectura Limpia:** Separación estricta entre la interfaz de usuario y la lógica del servidor para garantizar escalabilidad.
